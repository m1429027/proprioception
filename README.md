# Proprioception

本專案用於肩關節本體感覺測量。實驗時會將紅外線或紅色雷射筆固定在受試者身上，讓受試者將雷射光點打在投影幕上，再由攝影機即時偵測光點位置。系統先透過投影出的 ArUco / ChArUco 標記進行畫面校正，建立攝影機座標與投影幕真實位置之間的對應關係，之後即可記錄雷射點位置、計算位移距離，並將結果輸出成 `.txt` 檔。

## 專案目的

- 即時偵測投影幕上的雷射光點
- 將攝影機偵測到的座標轉換到投影幕座標
- 量測兩個雷射位置之間的距離
- 將量測結果輸出保存，供後續分析使用

## 功能介紹

- 使用 OpenCV 進行即時紅光偵測
- 使用投影幕上的 ArUco 標記進行校正
- 透過 homography 將相機畫面轉換到投影幕座標
- 提供曝光、對焦、亮度門檻、最小面積等可調參數
- 可記錄起點與終點並計算距離
- 可將像素距離換算成真實世界公分
- 可將量測結果輸出為 `.txt`

## 環境需求

- Python 3.9 以上
- Windows 環境
- 攝影機一台
- 投影幕或可投影平面
- 紅色或紅外線雷射筆

安裝套件：

```bash
pip install -r requirements.txt
```

`requirements.txt` 內容如下：

```txt
numpy>=1.21.0
opencv-contrib-python>=4.6.0.66
PyQt5>=5.15.0
```

## 檔案說明

- `main.py`：主程式，包含校正、雷射偵測、量測與輸出流程
- `calibration_data_cv.json`：儲存校正後的 homography matrix
- `requirements.txt`：套件需求

## 系統流程

### 1. 攝影機擷取

程式啟動後會開啟攝影機，持續讀取畫面。

### 2. 雷射光點偵測

程式會透過以下方式找出雷射光點：

- 灰階亮度門檻
- HSV 紅色區域遮罩
- 輪廓偵測與面積過濾

偵測到的最大亮點會視為目前雷射點位置。

### 3. 投影幕校正

按下 `c` 後，系統會進入校正模式，並在投影幕四角投出 4 個 ArUco 標記。攝影機偵測這些標記後，會計算 homography matrix，將攝影機畫面中的點映射到投影幕座標。

校正結果會儲存到：

```txt
calibration_data_cv.json
```

之後重新開啟程式時，若該檔案存在，會直接讀取先前的校正結果。

### 4. 距離量測

完成校正後：

- 第一次按 `Space`：記錄起點
- 第二次按 `Space`：記錄終點
- 程式會計算起點與終點之間的距離
- 並依目前比例尺換算成公分

畫面上也會顯示：

- 總距離
- 33% 位置
- 66% 位置

### 5. 結果輸出

按下 `s` 後，可將當前量測結果輸出成 `.txt` 檔。

輸出內容包含：

- 日期
- 時間
- 總距離
- 三分之一距離
- 三分之二距離

## 使用方式

執行：

```bash
python main.py
```

程式啟動後會出現兩個視窗：

- `Control Panel`：攝影機畫面與參數調整介面
- `Projector Screen`：投影畫面與量測顯示

## 操作按鍵

- `c`：進入校正模式，產生投影標記
- `Space`：記錄量測點
- `r`：重設目前量測
- `v`：修改比例尺對應的實際公分數
- `s`：輸出量測結果到 `.txt`
- `f`：切換投影幕全螢幕模式
- `q`：離開程式

## 建議操作流程

1. 執行 `python main.py`
2. 確認攝影機已正確開啟
3. 視需要調整曝光、對焦、亮度門檻與最小面積
4. 按下 `c` 進行投影幕校正
5. 校正完成後，確認 `calibration_data_cv.json` 已產生或更新
6. 讓受試者將雷射光點打到起始位置
7. 按下 `Space` 記錄起點
8. 讓受試者移動到目標位置
9. 再按一次 `Space` 記錄終點
10. 在投影畫面上確認量測結果
11. 按下 `s` 輸出 `.txt` 檔

## 輸出結果

### 校正輸出

- `calibration_data_cv.json`

內容為 homography matrix，用於後續座標轉換。

### 量測輸出

- 使用者指定路徑的 `.txt` 檔

輸出範例如下：

```txt
日期:2026/03/30 時間:14:25 總距離:25.3cm 三分之一:8.4cm 三分之二:16.9cm
```

## 注意事項

- 只要攝影機位置或投影幕位置改變，就建議重新校正
- 比例尺數值需要對應畫面中的實際長度，否則公分換算會失準
- 雷射偵測效果會受到曝光、對焦、環境光線與投影亮度影響
- 目前流程以單次起點到終點的量測為主，如需重測可按 `r` 重設
